Deershaper Seeds

Tienda online de cannabis

Descripción del Proyecto

Diseño web y material gráfico para Deershaper Seeds, marca especializada en la venta de semillas de cannabis.

El proyecto incluyó el desarrollo de su sitio web así como la creación de contenido visual para redes sociales (posts de Instagram) y cartelería promocional, manteniendo una identidad coherente y alineada con el estilo de la marca.

Proceso completo de diseño, desde wireframes iniciales hasta el diseño final.

Aprendizajes Clave

User Flow de una tienda online

Aprendí a definir y estructurar los recorridos que realiza un usuario dentro de una tienda online, desde la navegación inicial hasta la compra final. Este proceso me permitió entender mejor cómo guiar al usuario de forma lógica e intuitiva, optimizando cada paso para mejorar la experiencia y facilitar la conversión.
