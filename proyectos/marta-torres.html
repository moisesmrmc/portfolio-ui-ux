Branding Update para Marta Torres

Descripción del Proyecto

El proyecto consistió en renovar su branding personal para reflejar una imagen más coherente, actual y profesional.

A partir de un briefing inicial, se diseñó un logotipo trazado a mano, se redefinió la paleta de colores y se actualizó su presencia digital con el rediseño completo de su página web.

Además, se planteó una propuesta visual para la renovación de su perfil de Instagram, manteniendo coherencia estética en todos sus canales.

Herramientas Utilizadas

Adobe Photoshop Adobe Illustrator Figma Redes Sociales Instagram UI UX Responsive Design

El proyecto se llevó a cabo principalmente con Illustrator para el diseño de logotipo, colores y tipografías. También se hizo uso de Photoshop para retoque fotográfico y Figma para el rediseño de la Web y del perfil de Instagram
